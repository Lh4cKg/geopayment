# About
GeoPayment - Python SDK for Georgian Payment Providers

# Prerequisites
------

`Python >= 3.7.x`   
`cryptography >= 2.9.x`     
`pyOpenSSL >=19.x`


### Quick Start

Install **geopayment** package

```bash
$ pip install geopayment

```
